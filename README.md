# Smart Image Matcher

**Version:** 3.0.8 | **License:** GPL-2.0-or-later | **PHP:** 7.4+ | **WordPress:** 6.0+

Scans posts and pages for headings and inserts relevant media-library images next to each one.

Automatic updates for GitHub installs: see [`docs/RELEASE.md`](docs/RELEASE.md).
